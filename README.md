# Molecule Visualizer

A web application that converts molecular formulas to structural visualizations using PubChem database and RDKit.

## Features

- Convert molecular formulas to structural images
- Query PubChem database for compound information
- Generate SMILES notation
- Display 2D molecular structures

## Installation

1. Clone this repository
2. Create and activate venv environment:

```bash
python3 -m venv venv
source venv/bin/activate
```

3. Install all dependencies and run app.py

```bash
pip install -r requrements.txt
python3 app.py
```

4. Go to local web site a give some requests with common formuals("S2HO6", "CC")
